# Firebase Backend Adapter

This package is a [**Backend Adapter**](../../GLOSSARY.md#backend-adapter). It connects the main web application
to [Google's Firebase](https://firebase.google.com/) platform.

For definitions of technical terms, please see the main project [**Glossary**](../../GLOSSARY.md).

## Purpose

This adapter acts as a bridge to Firebase services. When the application needs to store data, manage users, or perform
other backend tasks, this package translates those requests into actions that Firebase can understand and perform.

Firebase is often used for rapid development and is a great choice for getting a project up and running quickly.

## Usage

This package is automatically used by the web application when it is configured to use Firebase as its backend provider.
