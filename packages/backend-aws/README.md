# AWS Backend Adapter

This package is a [**Backend Adapter**](../../GLOSSARY.md#backend-adapter). It connects the main web application
to [Amazon Web Services (AWS)](https://aws.amazon.com/).

For definitions of technical terms, please see the main project [**Glossary**](../../GLOSSARY.md).

## Purpose

This adapter acts as a bridge to AWS services. When the application needs to store data, manage users, or perform other
backend tasks, this package translates those requests into actions that AWS can understand and perform.

It is designed to be a powerful option suitable for large-scale, production use.

## Usage

This package is automatically used by the web application when it is configured to use AWS as its backend provider.
