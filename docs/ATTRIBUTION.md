# Attribution & Licensing

This project incorporates components and architectural patterns from **Notea**, an open-source markdown note-taking application.

**Original Project:**
- **Name:** Notea
- **Author:** [@qingwei-li](https://github.com/qingwei-li) (Cinwell Li)
- **Repository:** https://github.com/notea-org/notea
- **License:** MIT License

## Components Adapted
- S3-compatible storage providers (`src/lib/storage/*`)
- Rich markdown editor and related UI components (`src/components/editor/*`)
- Notes sidebar and tree UI (`src/components/notes/sidebar/*`)
- Notes state, hooks, and cache layers (`src/lib/notes/*`)
- i18n resources (`src/lib/i18n/*`, `src/locales/*`)
